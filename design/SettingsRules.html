HillsSim - Design - Setting Rules

The purpose of this file is to hold all the rules for the setting of the HillsSim program.

There are 3 types of settings: Primitive, Bounded, and Range.
Primitive, Bounded and Range settings are made from Basic values.
There are 2 types of Basic values: Unbuildable and Buildable.
Primitive settings can hold all basic values. Bounded and Range settings can only hold Buildable basic values.

Bounded values has a lower-bound and an upper-bound. For example, if a bounded variable X is between 1 and 10, then the lower-bound is 1 and the upper-bound is 10.
Range values has a start, end, and step size. For example, if a range variable X starts from 1, ends at 50, and has a step size of 1, then the range will be {1, 2, ..., 50}.

1. Table of Basic values

No. Name Type Description
1 String Unbuildable ASCII character set; List of characters used as text labels
2 Boolean Unbuildable True or False value
3 Integer Buildable Whole number; Similar to Integer data type
4 Float Buildable Number with 2 decimal points; For dollar amounts and percentage uses
5 Date Buildable ISO 8601 format: yyyy-mm-dd
6 Time Buildable ISO 8601 format: hh:mm:ss
7 DateTime Buildable ISO 8601 format: Combination of Date and Time values yyyy-mm-dd hh:mm:ss
